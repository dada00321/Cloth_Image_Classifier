Q： 為何不使用 2 個分類器(顏色、服飾分類) 串聯，減少製備訓練資料的時間成本？

A： 這樣的做法雖然直觀，預測效果也不錯，但辨識新資料、甚至辨識大批資料時，模型的延遲就可能大大降低使用者體驗了。
    
故採用此方法，將多個預測標籤放入同一模型。
